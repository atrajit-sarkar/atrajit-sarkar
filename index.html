$$\zeta(2) = \sum_{n=1}^{\infty} \frac{1}{n^{2}} = \frac{\pi^{2}}{6}$$

  • 抽象代数的視点でデータモデルを正規化

  • 系列・漸化式 → キャッシュ戦略最適化
  • 群論的抽象化 → API設計パターン

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print([fibonacci(i) for i in range(10)])
